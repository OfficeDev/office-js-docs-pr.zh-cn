# <a name="create-an-aspnet-office-add-in-that-uses-single-sign-on-preview"></a>创建使用单一登录的 ASP.NET Office 加载项（预览）

如果用户已登录 Office，加载项可以使用相同的凭据，这样用户无需重新登录，即可访问多个应用。有关概述，请参阅[在 Office 加载项中启用 SSO](sso-in-office-add-ins.md)。

## <a name="under-construction"></a>正在构建的内容

SSO API 最近存在更改。 我们正在进行新的演练，介绍如何创建使用 SSO 并通过 ASP.NET、OWIN 和适用于 .NET 的 Microsoft 身份验证库 (MSAL) 构建的加载项。 与此同时，请参阅 [Office 加载项 ASPNET SSO](https://github.com/officedev/office-add-in-aspnet-sso) 中的完整示例。
