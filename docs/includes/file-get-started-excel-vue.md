## <a name="update-the-app"></a><span data-ttu-id="67955-128">更新应用</span><span class="sxs-lookup"><span data-stu-id="67955-128">Update the app</span></span>

    <span data-ttu-id="67955-133">然后，在相同的位置上添加以下代码，并保存文件。</span><span class="sxs-lookup"><span data-stu-id="67955-133">Then add the following code in that same location, and save the file.</span></span>

4. <span data-ttu-id="67955-134">打开 **src/App.vue**，将文件内容替换为下列代码，在文件末尾（即 `</style>` 标记后）添加换行符，再保存此文件。</span><span class="sxs-lookup"><span data-stu-id="67955-134">Open **src/App.vue**, replace file contents with the following code, add a line break at the end of the file (i.e., after the `</style>` tag), and save the file.</span></span>

    ```html
      name: 'App',
      methods: {
        onSetColor () {
          window.Excel.run(async (context) => {
            const range = context.workbook.getSelectedRange()
            range.format.fill.color = 'green'
            await context.sync()
          })
        }
      }
    }
    </script>

    <style>
    #content-header {
        background: #2a8dd4;

        overflow: hidden;
    }

    #content-main {
        background: #fff;
        position: fixed;
        top: 80px;
        left: 0;
        right: 0;
        bottom: 0;
        overflow: auto;
    }

    .padding {
        padding: 15px;
    }
    </style>
    ```
