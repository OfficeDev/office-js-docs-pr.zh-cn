# <a name="alternateid-element"></a>AlternateId 元素

指定由 AppSource 发布的 Office 外接程序的备用 ID。

**加载项类型：** 内容、任务窗格和邮件

## <a name="syntax"></a>语法

```XML
<AlternateId>string </AlternateId>
```

## <a name="contained-in"></a>包含于

[OfficeApp](officeapp.md)

## <a name="remarks"></a>注解

您不会自己创建此值;将其提交到 AppSource 时, 会将其分配给你的外接程序。
