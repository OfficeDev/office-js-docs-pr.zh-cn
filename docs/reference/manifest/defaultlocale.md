# <a name="defaultlocale-element"></a>DefaultLocale 元素

指定外接程序中的字符串使用的区域设置的默认区域性名称。

**加载项类型：** 内容、任务窗格和邮件

## <a name="syntax"></a>语法

```XML
<DefaultLocale>string </DefaultLocale>
```

## <a name="contained-in"></a>包含于

[OfficeApp](officeapp.md)

## <a name="remarks"></a>注释

指定值采用 BCP 47 `language` 标记格式，如 **en-US**。
