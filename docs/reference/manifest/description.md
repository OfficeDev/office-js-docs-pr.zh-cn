# <a name="description-element"></a>Description 元素

指定“Office 加载项”的说明为不超过 250 个字符的字符串。

**加载项类型：** Content、Task pane、Mail

## <a name="syntax"></a>语法

```XML
<Description DefaultValue="string">
```
