# <a name="desktopsettings-element"></a>DesktopSettings 元素

指定在台式计算机上使用邮件外接程序时应用的源位置和控制设置。

**外接程序类型：** 邮件

## <a name="syntax"></a>语法

```XML
<DesktopSettings>
   ...
</DesktopSettings>
```
