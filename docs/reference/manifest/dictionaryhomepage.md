# <a name="dictionaryhomepage-element"></a>DictionaryHomePage 元素

指定该词典的主页的 URL。

**外接程序类型：** 任务窗格

## <a name="syntax"></a>语法

```XML
<DictionaryHomePage DefaultValue="URL" />
```
