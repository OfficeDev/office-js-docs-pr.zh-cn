# <a name="disableentityhighlighting-element"></a>DisableEntityHighlighting 元素

指定是否应为邮件外接程序关闭实体突出显示。

**外接程序类型：** 邮件

## <a name="syntax"></a>语法

```XML
<DisableEntityHighlighting> [true | false]</DisableEntityHighlighting>
```
