# <a name="equivalentaddins-element"></a>EquivalentAddins 元素

指定与等效的 COM 外接程序、XLL 或这两者的向后兼容性。

**外接类型：** 任务窗格，自定义函数

## <a name="syntax"></a>语法

```XML
<EquivalentAddins>
...  
</EquivalentAddins>  
```
