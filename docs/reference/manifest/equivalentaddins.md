# <a name="equivalentaddins-element"></a>EquivalentAddins 元素

指定与等效 COM 加载项和/或 XLL 的向后兼容性。

**外接程序类型：** 任务窗格、自定义函数

## <a name="syntax"></a>语法

```XML
<EquivalentAddins>
...  
</EquivalentAddins>  
```
