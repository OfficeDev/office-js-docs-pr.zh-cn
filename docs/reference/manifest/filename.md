# <a name="filename-element"></a>FileName 元素

指定您的 web 外接程序中的自定义函数的等效 XLL 的文件名。

**外接类型:** 自定义函数

## <a name="syntax"></a>语法

```XML
    <FileName>string</FileName>  
```
