# <a name="form-element"></a>Form 元素

在特定设备（台式机、平板电脑或手机）上运行时邮件加载项将使用的窗体的 UX 设置。

**加载项类型：** 邮件

## <a name="syntax"></a>语法

```XML
<Form>
   ...
</Form>
```
