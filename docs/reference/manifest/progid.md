# <a name="progid-element"></a>ProgId 元素

为 Web 加载项的任务窗格指定等效 COM 加载项的编程标识符。

**外接程序类型：** 任务窗格

## <a name="syntax"></a>语法

```XML
    <ProgId>string</ProgId>  
```
