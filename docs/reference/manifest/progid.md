# <a name="progid-element"></a>ProgID 元素

为 web 外接程序的任务窗格指定等效 COM 加载项的编程标识符。

**外接程序类型：** 任务窗格

## <a name="syntax"></a>语法

```XML
    <ProgID>string</ProgID>  
```
