# <a name="requestedwidth-element"></a>RequestedWidth 元素

指定内容 (的初始宽度) 以像素为单位）。

**外接程序类型：** 内容

## <a name="syntax"></a>语法

```XML
<RequestedWidth>integer</RequestedWidth>
```

## <a name="contained-in"></a>包含于

- [DefaultSettings](defaultsettings.md)（内容外接程序）：值可以在 32 至 1000 之间
