# <a name="targetdialect-element"></a>TargetDialect 元素

定义此字典支持的、表示为区域性名称字符串的区域语言。

**加载项类型：** 任务窗格

## <a name="syntax"></a>语法

```XML
<TargetDialect>
   string 
</TargetDialect>
```

## <a name="contained-in"></a>包含在

[TargetDialects](targetdialects.md)

## <a name="remarks"></a>备注

指定值采用 BCP 47 语言标记格式，如 `en-US`。
