# <a name="targetdialects-element"></a>TargetDialects 元素

定义区域语言或此字典支持的语言。

**外接程序类型：** 任务窗格

## <a name="syntax"></a>语法

```XML
<TargetDialects>
   ...
</TargetDialects>
```
