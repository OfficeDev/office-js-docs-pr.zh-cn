# <a name="outlook-javascript-api-requirement-sets"></a><span data-ttu-id="9617d-101">Outlook JavaScript API 要求集</span><span class="sxs-lookup"><span data-stu-id="9617d-101">Outlook JavaScript API requirement sets</span></span>

<span data-ttu-id="9617d-102">Outlook 加载项通过在其[清单](https://docs.microsoft.com/office/dev/add-ins/develop/add-in-manifests)中使用 [Requirements](/office/dev/add-ins/reference/manifest/requirements) 元素来声明所需要的 API 版本。</span><span class="sxs-lookup"><span data-stu-id="9617d-102">Outlook add-ins declare what API versions they require by using the [Requirements](/office/dev/add-ins/reference/manifest/requirements) element in their [manifest](https://docs.microsoft.com/office/dev/add-ins/develop/add-in-manifests).</span></span> <span data-ttu-id="9617d-103">Outlook 加载项始终包括 [Set](/office/dev/add-ins/reference/manifest/set) 元素，其中 `Name` 属性设置为 `Mailbox` 且 `MinVersion` 属性设置为支持加载项方案的 API 最低要求集。</span><span class="sxs-lookup"><span data-stu-id="9617d-103">Outlook add-ins always include a [Set](/office/dev/add-ins/reference/manifest/set) element with a `Name` attribute set to `Mailbox` and a `MinVersion` attribute set to the minimum API requirement set that supports the add-in's scenarios.</span></span>
