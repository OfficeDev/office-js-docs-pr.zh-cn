# <a name="powerpoint-and-project-in-the-javascript-api"></a>JavaScript API 中的 PowerPoint 和 Project

PowerPoint 和 Project 支持通过 JavaScript API 创建的加载项。 但是，它们当前没有特定于主机的 API。 你可以通过通用 API 与这些主机交互。
